# 🎯 用户体验流程设计
## Web3固定收益平台核心用户旅程

> **设计目标**：将复杂的Web3操作简化为直观的用户流程，确保每个关键环节都有清晰的指引和反馈

## 1. 🚀 用户注册购买流程

### 主流程设计
```
Landing Page → 连接钱包 → 选择卡片 → 确认购买 → 获得NFT → 查看收益
```
